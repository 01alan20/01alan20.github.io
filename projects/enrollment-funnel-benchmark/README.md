# Enrollment Funnel Model

Synthetic + benchmark hybrid pipeline for inquiry to enrollment analysis.

## Local Data Files
- `data/enrollment_funnel_synthetic.csv`
- `data/enrollment_funnel_summary.csv`

## Intended Dashboard Tabs
- Executive Summary
- Funnel Conversion
- Source and Market Mix
- Conversion Velocity
- Segment Drilldown

## Source Note
Synthetic student-level data calibrated to realistic higher-ed funnel ranges. Benchmark guardrails are intentionally kept within the user-defined ranges: inquiry to application 20% to 40%, application to offer 60% to 80%, and offer to enrollment 20% to 50%.
