# B2B Sales Pipeline Dashboard

Static GitHub Pages dashboard.

## What changed in this version

- Opens directly into the dashboard.
- Includes Executive Summary, Product & Revenue Analysis, Performance Over Time, Sales Team Performance, Open Pipeline Prioritization, and Growth Questions & Actions.
- Adds new opportunities/leads by quarter.
- Adds conversion by quarter.
- Adds product conversion by quarter.
- Adds growth questions and answers tied to Maven revenue and pipeline decisions.

## GitHub Pages setup

Upload the contents of this folder to your repo root, or put them in `/docs` if your Pages source is `/docs`.

Use `index.html` as the page entry point.

## Data note

This dataset is opportunity pipeline data. “New leads” in the dashboard means newly engaged opportunities based on engage date.
