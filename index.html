Singapore Condo Investment ROI

HTML • JavaScript • Plotly.js | 2025

A browser-only tool that lets investors explore how condo purchase prices and monthly rents have moved from 2020 to 2025 and what that means for ROI. It loads a simple pmi_roi_agg.csv at runtime, so the page stays 100 % static and deploys on GitHub Pages with no backend.

Key Features

  • CSV parsed in-browser with Papa Parse — no server needed
  • Plotly dual-axis chart (buy price & rent) that updates instantly
  • Controls: project dropdown, size slider, budget input
  • Budget line shows how many units you can afford today
  • Fully responsive & green-tick Lighthouse score
